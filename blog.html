Important FAQ ❔

What is the Document Object Model(DOM)?

The DOM is a tree-like representation of a webpage's structure that allows dynamic access and manipulation of its content using JavaScript.

How do you select an element from the DOM?

1.document.getElementById() – Selects an element by its ID.
2.document.querySelector() – Selects the first element that matches a CSS selector.
3.document.querySelectorAll() – Selects all elements that match a CSS selector (returns a NodeList).
and many more..

What is event delegation in the context of the DOM and Why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to handle events from its child elements, improving performance and supporting dynamic content.

How do you manipulate an element's attribute and styles using the DOM?

To manipulate an element's attributes and styles: Attributes: element.setAttribute('attr', 'value') or element.attr = 'value' Styles: element.style.property = 'value' or use element.classList to add/remove classes.
